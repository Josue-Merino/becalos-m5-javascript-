# Intro a Intro a Vite
Vite se ha convertido en una herramienta clave en el desarrollo moderno de aplicaciones web, permitiendo a los desarrolladores trabajar de manera más eficiente y con tiempos de carga reducidos. Su compatibilidad con múltiples frameworks y su enfoque en la velocidad lo hacen una excelente opción tanto para principiantes como para expertos en desarrollo frontend y Node.js.
La solución a este ejercicio esta dividida en el archivo `main.js` e `index.html`.

## Objetivo

Aprender a crear un proyecto con Vite, comprender su estructura básica y practicar el uso de Vanilla JS para manipular el DOM y crear interactividad en una página web a través de un formato de 'workshop/taller'. Al final de este taller, serás capaz de:

1. Instalar y configurar un proyecto con Vite.
2. Comprender la estructura básica de un proyecto de Vite con Vanilla JavaScript.
3. Manejar eventos en JavaScript para actualizar contenido dinámico en la página.
4. Utilizar ES Modules para organizar el código de manera eficiente.

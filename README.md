# Bécalos Tech Challenge - Módulo 5: Desarrollo Avanzado en JavaScript

Este repositorio está dedicado a recopilar todos los ejercicios y proyectos desarrollados durante el **Módulo 5** del programa **Bécalos Tech Challenge**. El enfoque principal de esta etapa es el dominio profundo de JavaScript, abarcando desde las características modernas de ECMAScript, estructuras de datos y algoritmos avanzados, hasta el manejo del entorno asíncrono, consumo de APIs y desarrollo frontend moderno.

## 📂 Organización del Repositorio

Para facilitar la navegación y el aprendizaje, el repositorio se organiza bajo la siguiente estructura:

* **Carpetas:** Cada tema o proyecto cuenta con su propio directorio independiente.
* **Archivos:** Dentro de cada carpeta se encuentran los archivos correspondientes de cada solución según sea el caso (`.html`, `.css`, `.js`).
* **Documentación Local:** Cada ejercicio incluye su propio archivo `README.md` detallando los objetivos, explicaciones técnicas y enfoques de resolución.
